## Requirements
[Node.js](https://nodejs.org/) v16 Current (16.3) or greater

## Quick Start
**Clone** the repository and run
```bash
$ npm install
```

**Setup** a `.env.local`
```env
NEXT_PUBLIC_TOKEN=*Endpoint token*
NEXT_PUBLIC_GRAPHQL_URL=*Graphql End-point url*
```

**Run** the project!
```bash
$ npm start
```
